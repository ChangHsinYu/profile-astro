# Gatsby src to Astro

這是從原本 Gatsby `src` 轉成的 Astro 專案。

## 操作方式

1. 安裝套件

   ```bash
   npm install
   ```

2. 開發預覽

   ```bash
   npm run dev
   ```

   開啟終端機顯示的本機網址，通常是 `http://localhost:4321`。

3. 正式建置

   ```bash
   ASTRO_TELEMETRY_DISABLED=1 npm run build
   ```

4. 預覽建置結果

   ```bash
   npm run preview
   ```
